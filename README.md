# Formation Réseaux & Cybersécurité

Formation live en ligne, environnements émulés, labs pratiques et explications professionnelles.  

Ce site a été créé à l’aide des technologies **HTML, CSS et JavaScript**.  
Le site a été vérifié et amélioré à l’aide d’outils d’intelligence artificielle pour garantir qualité et performance.

## Objectif

Le but est de créer un site où tout le monde peut voir les formations que j’offre.  
- **Langue principale :** Français  
- **Diffusion des formations :**  
  - Live sur **Discord** avec une haute qualité  
  - Ou sur **Google Meet**
